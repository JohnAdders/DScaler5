Audio Decoder Filter
 

This filter can decode all types of DVD and DVB audio including MPEG layers 1 2 & 3, DTS, AC3 and PCM.

There are a few settings available to configure the decoder and these are explained below.

Setting Default Description
Speaker Config Stereo Dolby Downmix Select the speaker configuration that most closely matches your owm.
Dynamic Range Control On This option limits the dynamic range for DTS and AC3 streams. With PC speakers leave this on, for use with Home Theater speakers switch off.
Use SP/DIF Off Switch on to pass through AC3 and DTS streams via SP/DIF.
Mpeg Over SP/DIF Off Allows sending MPEG audio streams over SP/DIF. This mode may only work with most sound cards at a sample rate of 48kHz and so is not suitable for use with MP3s taken from CDs.
SP/DIF Audio Time Offset 0 Allow correction of the delay on SP/DIF, negative values are allowed.
