How to use the WTW finder
=========================

Expand all the files in the zip to a place where you have write access.

To analyse the decoded yuv content start the wtwfindyuv.cmd batch file.

To analyse the decoded yuv content start the wtwfindrgb.cmd batch file.

The 2 batch files can be started with a paramter which is the file you want to 
analyse or they can be ruin interactively and you will be asked for the file you 
wish to analyze.
